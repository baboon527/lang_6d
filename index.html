Learning 6-DoF Object Poses to Grasp Category-Level Objects by Language

Chilam Cheang
Haitao Lin
Yanwei Fu
Xiangyang Xue

Fudan University




Abstract

Given an RGB-D image and a language description, we first ground target 2D position in the image plane, and infer the 6-DoF pose of the target object. Finally, robotic grasping is implemented by utilizing the estimated 6-DoF object pose.
